# 中断和异常  

操作系统夺回cpu使用权的唯一方式就是中断

    内中断:比如应用程序企图使用特权指令,或者数组越界需要处理(转到 OS 处理异常),
        或者陷入指令(应用程序希望 OS 为他提供某些操作),陷入指令不是特权指令
    外中断:时钟中断(实现并发),I/O中断请求等等
